# DevOpsZealot Example: Adding S3 Bucket

This example shows how to use DevOpsZealot to automatically add an S3 bucket to your Terraform configuration.

## Setup

1. Make sure DevOpsZealot server is running:
   ```bash
   python -m zealot.server
   ```

2. Set your API key (if configured):
   ```bash
   export ZEALOT_API_KEY=your-api-key
   ```

## Example Request

```bash
curl -X POST http://localhost:8090/api/v1/tasks \
  -H "Content-Type: application/json" \
  -H "X-Zealot-API-Key: ${ZEALOT_API_KEY}" \
  -d '{
    "repository": "https://github.com/yourusername/infrastructure.git",
    "branch": "main",
    "files": ["terraform/s3.tf"],
    "requirements": [
      "Add a new S3 bucket named company-logs-2025",
      "Enable versioning on the bucket",
      "Add lifecycle policy to delete objects after 90 days",
      "Tag with Environment=production and ManagedBy=zealot"
    ],
    "validation_rules": [
      "terraform_plan",
      "security_scan"
    ]
  }'
```

## Expected Result

DevOpsZealot will:
1. Clone your repository
2. Create a feature branch `zealot/xxxxx`
3. Read the current `terraform/s3.tf` file
4. Use AI to add the S3 bucket configuration
5. Validate the changes with `terraform plan`
6. Run security scans
7. Commit and push the changes
8. Create a pull request

## Example Generated Code

```hcl
# terraform/s3.tf - Generated by DevOpsZealot

resource "aws_s3_bucket" "company_logs_2025" {
  bucket = "company-logs-2025"
  
  tags = {
    Environment = "production"
    ManagedBy   = "zealot"
  }
}

resource "aws_s3_bucket_versioning" "company_logs_2025" {
  bucket = aws_s3_bucket.company_logs_2025.id
  
  versioning_configuration {
    status = "Enabled"
  }
}

resource "aws_s3_bucket_lifecycle_configuration" "company_logs_2025" {
  bucket = aws_s3_bucket.company_logs_2025.id
  
  rule {
    id     = "delete-after-90-days"
    status = "Enabled"
    
    expiration {
      days = 90
    }
  }
}
```

## Check Status

```bash
# Get task status
curl http://localhost:8090/api/v1/tasks/{task_id} \
  -H "X-Zealot-API-Key: ${ZEALOT_API_KEY}"
```

## Python Client Example

```python
import requests
import time

# Submit task
response = requests.post(
    "http://localhost:8090/api/v1/tasks",
    json={
        "repository": "https://github.com/yourusername/infrastructure.git",
        "files": ["terraform/s3.tf"],
        "requirements": ["Add S3 bucket for logs with 90-day retention"]
    },
    headers={"X-Zealot-API-Key": "your-api-key"}
)

task_id = response.json()["task_id"]
print(f"Task submitted: {task_id}")

# Poll for completion
while True:
    status = requests.get(
        f"http://localhost:8090/api/v1/tasks/{task_id}",
        headers={"X-Zealot-API-Key": "your-api-key"}
    ).json()
    
    print(f"Status: {status['status']}")
    
    if status['status'] in ['completed', 'failed']:
        if status['result']:
            print(f"PR URL: {status['result']['pr_url']}")
        break
    
    time.sleep(5)
```
